BTech CSE Curriculum Overview

1st Semester

  • Introduction to Programming: Basics of C or Python programming.
  • Mathematics I: Calculus, linear algebra, matrices.
  • Engineering Physics: Mechanics, thermodynamics, waves.
  • Basic Electrical Engineering: Electrical circuits, Ohm's law.
  • Engineering Graphics: Technical drawing, CAD.
  • Communication Skills: Technical writing, public speaking.

2nd Semester

  • Data Structures: Arrays, linked lists, trees, graphs.
  • Discrete Mathematics: Logic, set theory, graph theory.
  • Digital Logic Design: Boolean algebra, logic gates.
  • Engineering Chemistry: Chemical bonding, polymers.
  • Object-Oriented Programming: Classes, objects, inheritance.
  • Environmental Science: Ecosystems, pollution control.

3rd Semester

  • Algorithms: Analysis, sorting, searching, dynamic programming.
  • Computer Organization: CPU architecture, memory hierarchy.
  • Operating Systems: Process management, file systems.
  • Database Management Systems: SQL, database design.
  • Software Engineering: SDLC, design patterns.
  • Probability and Statistics: Probability theory, statistical inference.

4th Semester

  • Computer Networks: OSI model, TCP/IP, routing.
  • Design and Analysis of Algorithms: Graph algorithms, complexity theory.
  • Microprocessors: Architecture, assembly language programming.
  • Theory of Computation: Automata theory, Turing machines.
  • Web Technologies: HTML, CSS, JavaScript, web servers.
  • Human-Computer Interaction: User interface design, usability.

5th Semester

  • Artificial Intelligence: Search algorithms, machine learning basics.
  • Compiler Design: Lexical analysis, syntax analysis.
  • Data Communication: Data transmission, error detection.
  • Mobile Computing: Mobile networks, wireless communication.
  • Elective I: Cloud computing, big data, cybersecurity.
  • Minor Project: Small-scale system or application development.

6th Semester

  • Machine Learning: Supervised, unsupervised learning, neural networks.
  • Software Testing: Testing techniques, test automation.
  • Distributed Systems: Distributed computing, fault tolerance.
  • Computer Graphics: 2D/3D graphics, rendering techniques.
  • Elective II: IoT, advanced databases, robotics.
  • Major Project - Phase I: Research, design, initial implementation.

7th Semester

  • Big Data Analytics: Data mining, Hadoop, data visualization.
  • Cloud Computing: Service models, virtualization, cloud security.
  • Cybersecurity: Network security, cryptography.
  • Elective III: Blockchain, advanced AI, bioinformatics.
  • Major Project - Phase II: Project completion, testing, presentation.
  • Industry Internship: Practical industry experience.

8th Semester

  • Internship and Project Work: Extensive project focusing on real-world applications.
  • Professional Elective IV: Quantum computing, NLP, advanced AI.
  • Professional Elective V: Cybersecurity, cloud infrastructure, data science.
  • Research Methodology: Research techniques, data analysis, thesis preparation.
  • Seminar: Current trends and technologies in computer science.
